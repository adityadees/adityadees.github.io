Web Development

  Dalam merancang dan mengembangkan suatu website agar dapat digunakan dengan baik dan berjalan sebagai mana mestinya haruslah memiliki kemampuan visual dan logika yang tajam. Web Developer adalah seseorang yang menciptakan aplikasi berbasis web dengan menggunakan bahasa pemrograman. Tugas dari Web Developer ini menjadi penghubung dari seluruh sumber daya yang dibutuhkan untuk pembuatan website. Tugas mereka akan berakhir jika website yang dibuat sudah siap untuk digunakan dan tidak terdapat masalah dalam penggunaan website tersebut.
  Web developer umumnya dibagi menjadi dua yaitu Frontend dan Backend Developer, namun jika orang tersebut mengerjakan frontend dan backend sekaligus maka dapat disebut sebagai Full Stack Developer. Seorang Full Stack Developer bertanggung jawab atas seluruh hal, mulai dari tampilan hingga ke logika fungsional. Saya sudah menjadi fullstack developer selama lebih dari 5 tahun, namun jika kerja team terkadang saya menjadi frontend dan terkadang juga saya menjadi backend.
