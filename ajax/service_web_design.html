Web Design

  Web design adalah suatu kegiatan yang bertujuan untuk merancang aplikasi dan tampilannya. Orang yang melakukan kegiatan tersebut dikenal dengan sebutan web designer. Peran dari seorang web designer adalah merancang mulai dari prototype hingga mengintegrasikan semua komponen dan semua sumber daya yang dipakai untuk membuat sebuah rancangan website seperti membuat tampilan halaman aplikasi web yang dinamis dan user friendly, mengatur cara para pengunjung berinteraksi dengan semua elemen yang ada dalam website, dan lain sebagainya. Jika seorang web designer sudah handal dan berpengalaman, dia akan terbiasa dengan berbagai tampilan web baik dari sisi client-side scripting atau sisi client. Untuk membuat desain tersebut, mereka bisa menggunakan aplikasi desain yang menunjang pembuatan web design. Sebagai seorang fullstack developer sayapun biasanya merangkap semunya mulai dari prototype hingga melakukan desain langsung sesuai dengan apa yang diinginkan oleh client sebelum dilanjutkan ke tahap selanjutnya.
